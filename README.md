# 🔐 Assessment_01 – React MUI Dashboard Application

A modern React application featuring a **Login Screen** and a **Dashboard Layout with Left Drawer Navigation**, built using **React + TypeScript + Vite**.

---

## 🚀 Features

### ✅ Authentication UI

- Clean and responsive Login Screen
- Built using Material UI components
- Ready for backend authentication integration

### ✅ Dashboard Layout

- Persistent left-side navigation drawer
- Toggleable sidebar
- Responsive layout structure
- Nested routing using React Router v6

### ✅ Routing Structure

| Route   | Description  |
| ------- | ------------ |
| `/`     | Login Screen |
| `/home` | Home Page    |

---

## Tech Stack

- ⚛️ React
- 🟦 TypeScript
- ⚡ Vite
- 🧭 React Router v6
- 🎨 Material UI (MUI v5)
- 💅 Emotion Styled Components
